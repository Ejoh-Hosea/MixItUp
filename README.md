A React application for browsing and searching cocktails using the Cocktails db API.

## features

- React router v6 (nested routes, loaders, actions)
- React query with caching + prefetching
- Axios for data fetching
- Search with query params
- Dynamic single-cocktail pages
- Styled-components for responsive ui
- React-toastify for notifications
- Custom error and loading states
- Full integration with the cocktails db api

## tech stack

- React
- React router v6
- @tanstack/react-query
- Axios
- Styled-components
- React-toastify

## installation

```bash
npm install
npm run dev
```
